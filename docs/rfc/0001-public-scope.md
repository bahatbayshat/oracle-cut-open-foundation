# RFC 0001: Public Scope

ORACLE CUT Open Foundation is the public, open-source layer for reusable safety and verification patterns in AI-assisted creative workflows.

## Public components

- Prompt governance patterns
- Human approval gates
- Verification report structures
- Metadata provenance examples
- Documentation templates
- Safe automation boundaries

## Non-public components

This repository does not include private production systems, provider credentials, proprietary media pipelines, commercial workflow logic, or unpublished ORACLE CUT internals.

## Goal

The goal is to help developers build safer AI creative tooling with clear approval boundaries, auditable reports, and reusable engineering practices.
